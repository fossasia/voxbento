The Voxbento Admin Panel (often referred to as Mission Control) is the central hub for managing your interpretation infrastructure. Depending on your role (`super_admin` or `event_owner`), your view of the dashboard will be tailored to what you have permission to manage.

## 1. Global Navigation (Sidebar)

The left sidebar is your primary navigation tool. 

- **Dashboard**: The home screen, showing at-a-glance metrics for all your events.
- **Events**: The master list of all events. `super_admin` users will see an "Add Event" button here.
- **Users**: (Restricted to `super_admin`). Manage registered accounts and grant site-wide admin privileges.
- **API Settings**: Manage the global API keys for Transcription providers (like OpenAI or Deepgram).

## 2. Event Summary Cards

On the main dashboard, each event you have access to is displayed as a summary card.

- **Live Booths Metric**: Shows a real-time count of how many booths currently have an `active_interpreter` broadcasting audio.
- **Rooms & Booths Count**: A quick summary of the total configured infrastructure for that event.
- **Manage Button**: Clicking this takes you deep into the event's configuration page to add rooms or configure language channels.

## 3. System Health Diagnostics

At the top of the dashboard, you will see a critical health indicator for the media server.

- **MediaMTX Reachable (Green)**: The backend portal can successfully communicate with the MediaMTX ingest server. WHIP and WHEP streams will work normally.
- **MediaMTX Unreachable (Red)**: The portal cannot connect to the media server. **Interpreters will not be able to go live.** If you see this, contact your system administrator to check the Docker container status.

## Role Differences in the UI

Voxbento strictly enforces what you can see and click based on your role:

- **`super_admin`**: Can see the "Users" tab, can click "Add Event", and sees every event on the platform.
- **`event_owner`**: The "Users" tab is hidden. The "Add Event" button is hidden. You will only see the summary cards for events that you have been explicitly assigned to manage.
